# AIxEdu

AIxEdu is a one-page personal website focused on technology, education, AI, STEAM, and Maker education.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- lucide-react
- framer-motion

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```
